## 사용방법
텍스트를 선택(블록 지정) 후 <kbd>Ctrl</kbd> + <kbd>Command</kbd> + <kbd>k</kbd> 키를 누르면 해당 텍스트의 한글 맞춤법 검사를 실행합니다. 실행 시 웹브라우저를 통해 [부산대 한국어 맞춤법/문법 검사기](http://speller.cs.pusan.ac.kr/) 사이트로 연결됩니다.
